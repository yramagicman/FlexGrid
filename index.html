Choosy

The responsive grid system that lets you decide

Why Choose the Choosy Grid System

As far as I know, outside of Twitter's Bootstrap, this is the only responsive grid system that incorporates the ability to specifically choose what is shown and hidden and how your page breaks. So if you want your page to look one way on an iPad, another on a mobile phone, and yet another on desktops and laptops, this system makes it possible. And while at the moment, this page is entirely lorem ipsum text, I think it might give you an idea of how the system works

If you're curious, check out an example, or keep reading to find out more.

Math and Media Queries

The Choosy Grid System is percentage based, not pixel based, so you don't have to bother with nasty re-calculations when changing media types. Just add the modify theappropriate css file and let the system take do the work. This also allows for completely fluid layouts, or adaptive layouts, if that's more your thing. Just swich the media query file when you want to switch responsive modes.

Built in compression

This grid system contains a buit in PHP based CSS compression script that compiles and minifies all the necessary CSS files. All you have to do is include your css files along with the others and they're automatically compressed with the rest of the files. just use
<link href="example-folder/minified.php" type="text/css" rel="stylesheet">
instead of the usual link to a stylesheet. The cool thing about this system, is that your stylesheets are always readable and editable, but you serve the end user a slower page because they get a compressed file.

Typography and Images

Like most other grid systems, this one comes with basic typography and image styles already defined, saving you time.

WordPress Classes

The Choosy Grid System also comes with a stylesheet containing all the classes required to make a valid WordPress theme and a few more usefull classes so that you don't have to go sniffing through the code of your theme to find an obscure comment or widget class.
